# automacao-de-mesclagem
🚀 Transformando 1 hora de trabalho manual em poucos segundos com automação

Recentemente desenvolvi uma solução em Python para automatizar um processo operacional envolvendo tratamento e cruzamento de dados em Excel.

O desafio era claro: consolidar informações de diferentes bases, aplicar filtros específicos e enriquecer os dados com informações adicionais — algo que normalmente levaria bastante tempo e estaria sujeito a erros manuais.

💡 A solução:

* Leitura automatizada de múltiplas planilhas Excel
* Aplicação de regras de negócio (filtros como transportadora e desvio logístico)
* Cruzamento de dados (equivalente ao PROCV)
* Tratamento de inconsistências e duplicidades
* Interface simples para o usuário (sem necessidade de conhecimento técnico)
* Geração de arquivo final pronto para uso em dashboard

🔧 Tecnologias utilizadas:
Python | Pandas | Tkinter | PyInstaller

📦 Diferencial:
O projeto foi convertido em um executável (.exe), permitindo que qualquer pessoa utilize a ferramenta com apenas um clique — sem necessidade de instalar Python.

📊 Resultado:
Um processo que levava cerca de 1 hora passou a ser executado em segundos, com mais confiabilidade e padronização.

Esse tipo de solução mostra como automação e análise de dados podem gerar impacto direto na operação.
